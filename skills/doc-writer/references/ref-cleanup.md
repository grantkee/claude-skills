# Cleanup pass

Used by the cleanup agent (Wave 5). Read `tech-doc-rules.md` first.

Your sole job: remove conversational artifacts using the four rules below. Do not make any other changes.

## 1. No conversational sign-offs

Delete any of these (and similar patterns):
- "I hope this helps!"
- "Let me know if you have any questions!"
- "Feel free to reach out if you need anything else!"
- "Happy to help further!"
- "Hope that clarifies things!"
- "Don't hesitate to ask!"

These are chatbot artifacts. In a document, they make no sense. In a conversation, they waste the reader's time.

## 2. No knowledge disclaimers

Delete "as of my last update," "based on available information," "to the best of my knowledge," and similar hedges from documents. If the document states a fact, state it as fact. If you are genuinely uncertain about something specific, say what you are uncertain about.

## 3. No sycophancy

Delete "Great question!", "Excellent point!", "That's a really interesting observation!" and similar. Just answer the question or continue the document.

## 4. No meta-commentary

Delete narration of what is about to happen. Do not write "Here is a summary of the changes" before a summary, or "Below you will find a detailed explanation" before an explanation. Just write the summary. Just write the explanation.

Specific patterns to delete or rewrite:
- "Here is a/an..."
- "Below you will find..."
- "The following section describes..."
- "In this document, we will..."
- "Let's take a look at..."
- "As mentioned above/below..."
